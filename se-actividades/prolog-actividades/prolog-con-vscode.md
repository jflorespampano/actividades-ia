# prolog y vscode

Para programar en prolog desde vescode

## opcion 1 

### Verifica si esta instalado swipl

```bash
swipl --version
```

### Cargar y consultar tu código usando el terminal integrado

1. Abre un archivo `prueba.pl` en VSCode 
2. Abre el terminal integrado con Ctrl+ñ
3. Inicia SWI-Prolog, escribe: `swipl`, devolvera el promp: `?-`
4. carga tu archivo `?- [prueba].`, sin extensión
5. ejecuta tus consultas.
6. si modificas y guardas el archivo, en la consola escribe: `make.`
7. para salir del interprete escribe: `halt.`
