## Perplexity

¿Qué es Perplexity AI?
Motor de búsqueda conversacional: Permite hacer preguntas en lenguaje natural (como ChatGPT) pero con resultados enlazados a fuentes web actualizadas.

Enfoque en precisión: Responde con citas y enlaces a sitios confiables (evitando "alucinaciones" típicas de otros LLMs).

Disponible en web y app móvil (iOS/Android).

## Herramientas

* Weka es una suite de software de código abierto (open-source) especializada en aprendizaje automático (Machine Learning) y minería de datos. Su nombre viene del ave *Weka* (endémica de Nueva Zelanda, donde se desarrolló la herramienta) y también es un acrónimo de "Waikato Environment for Knowledge Analysis" (Entorno Waikato para Análisis de Conocimiento), creado por la Universidad de Waikato.

## Entrega de actividades

Deverán crear un repositorio en Github donde cargarán sus actividades de entrega y enviarán a la tarea asignada en Teams, la liga de su repositorio.
La carpeta de actividades tendrá un archivo `readme.md` que describe los datos del/los integrantes, matricula, nombre, materia, profesor. Cada actividad a entregar deberá tener su propia subcarpeta y un archivo `readme.md` que describe la actividad.
La estructura de las carpetas de entrega en el repositorio será:

```text
ia/
├── recpat/    # Reconocimiento de Patrones
├── se/        # Sistemas Expertos
├── alg/       # Algoritmos de IA
└── pp/        # Programación en paralelo
    readme.md
```

## Empezar

>**Nota:** recibira estos materiales del profesor: desde GitHub una liga donde clonas/bajas la carpeta *actividades-ia* y el profesor te proporcionará dos archivos `zip`: uno con la carpeta `ia` y el otro con una página web, en este curso nos referiremos a estos 3 elementos como: carpeta *actividades-ia*, carpeta *ia* y *página web*.

Para empezar realice lo siguiente:

1. Descargue la carpeta  `actividades-ia` de Github
2. De la carpeta `ia/paginaWeb` descomprima el archivo `pag_web.zip`
3. Realice la actividad: `actividades-ia/se-actividades/se-actividad-01.MD` 
4. Instala git y crea una cuenta de GitHub apoyate en: `git.md`
5. Al terminar realiza la actividad `actividades-ia/se-actividades/actividad01.arboles/actividades.MD`
